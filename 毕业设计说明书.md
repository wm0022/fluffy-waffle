# 圣惟书店管理系统设计与实现

## 摘要

随着互联网技术的快速发展和电子商务的普及，传统书店的经营模式面临着巨大的挑战。为了提高书店的管理效率、降低运营成本、提升用户体验，本文设计并实现了一套基于Web的圣惟书店管理系统。

本系统采用前后端分离的架构，后端使用Spring Boot框架，前端使用Vue.js框架，数据库采用MySQL，缓存采用Redis。系统实现了图书管理、订单处理、捐赠管理、会员管理、库存管理、服务反馈等核心功能，同时具备完善的用户认证与授权机制，确保系统的安全性。

本文首先介绍了项目的背景和意义，然后对相关技术进行了概述，接着进行了系统需求分析和设计，最后详细阐述了系统的实现过程，并进行了系统测试。测试结果表明，本系统运行稳定，功能完善，能够满足书店的日常管理需求。

**关键词**：书店管理；Spring Boot；Vue.js；MySQL；Redis

## Abstract

With the rapid development of Internet technology and the popularity of e-commerce, the business model of traditional bookstores is facing great challenges. In order to improve the management efficiency of bookstores, reduce operating costs, and enhance user experience, this paper designs and implements a Web-based Shengwei Bookstore Management System.

The system adopts a front-end and back-end separated architecture. The back-end uses the Spring Boot framework, the front-end uses the Vue.js framework, the database uses MySQL, and the cache uses Redis. The system implements core functions such as book management, order processing, donation management, member management, inventory management, and service feedback. At the same time, it has a complete user authentication and authorization mechanism to ensure the security of the system.

This paper first introduces the background and significance of the project, then outlines the related technologies, then conducts system requirement analysis and design, finally elaborates on the system implementation process in detail, and conducts system testing. The test results show that the system runs stably, has complete functions, and can meet the daily management needs of bookstores.

**Keywords**: Bookstore Management; Spring Boot; Vue.js; MySQL; Redis

## 1 绪论

### 1.1 课题背景

随着互联网技术的快速发展和电子商务的普及，传统书店的经营模式面临着巨大的挑战。一方面，网络购书平台的兴起给传统书店带来了巨大的竞争压力；另一方面，传统书店的管理方式相对落后，存在着效率低下、成本较高、信息更新不及时等问题。

为了应对这些挑战，提高书店的管理效率、降低运营成本、提升用户体验，开发一套功能完善、操作便捷、安全可靠的书店管理系统显得尤为重要。

### 1.2 课题意义

本课题的研究意义主要体现在以下几个方面：

1. **提高管理效率**：通过信息化手段，实现图书管理、订单处理、库存管理等业务流程的自动化，减少人工操作，提高管理效率。
2. **降低运营成本**：通过系统的优化设计，减少人力、物力等资源的消耗，降低运营成本。
3. **提升用户体验**：提供便捷的在线购书、捐赠申请、会员服务等功能，提升用户的满意度和忠诚度。
4. **促进管理决策**：通过系统的数据分析功能，为书店的经营决策提供数据支持。

### 1.3 国内外研究现状

#### 1.3.1 国外研究现状

国外在书店管理系统方面的研究起步较早，已经形成了较为成熟的技术和产品。例如，亚马逊、巴诺等大型图书电商平台，拥有完善的图书管理、订单处理、物流配送等系统，技术水平处于领先地位。

同时，国外也有许多中小型书店采用了专业化的书店管理软件，这些软件功能完善，操作便捷，能够满足中小型书店的管理需求。

#### 1.3.2 国内研究现状

国内在书店管理系统方面的研究起步较晚，但发展迅速。近年来，随着电子商务的发展，国内出现了许多优秀的图书电商平台，如当当、京东图书等，这些平台拥有完善的图书管理、订单处理、物流配送等系统。

同时，国内也有许多针对传统书店的管理软件，这些软件功能相对简单，主要满足图书的进销存管理需求。但随着技术的发展，越来越多的书店管理软件开始注重用户体验和数据分析功能。

## 2 相关技术介绍

### 2.1 后端技术

#### 2.1.1 Spring Boot

Spring Boot是由Pivotal团队提供的一个全新框架，其设计目的是用来简化新Spring应用的初始搭建以及开发过程。该框架使用了特定的方式来进行配置，从而使开发人员不再需要定义样板化的配置。

Spring Boot具有以下特点：
- 独立运行的Spring应用程序
- 内嵌Tomcat、Jetty等Servlet容器
- 提供自动配置的“starter”依赖
- 尽可能地自动配置Spring
- 提供生产就绪功能，如指标、健康检查和外部配置
- 绝对没有代码生成，不需要XML配置

在本系统中，后端采用Spring Boot框架，简化了开发过程，提高了开发效率。

#### 2.1.2 MyBatis Plus

MyBatis Plus是一个MyBatis的增强工具，在MyBatis的基础上只做增强不做改变，为简化开发、提高效率而生。

MyBatis Plus具有以下特点：
- 无侵入：只做增强不做改变，引入它不会对现有工程产生影响
- 损耗小：启动即会自动注入基本CRUD，性能基本无损耗
- 强大的CRUD操作：内置通用Mapper、通用Service，仅通过少量配置即可实现单表大部分CRUD操作
- 支持Lambda形式调用：通过Lambda表达式，方便的编写各类查询条件
- 支持主键自动生成：支持多达4种主键策略
- 内置代码生成器：采用代码或Maven插件可快速生成Mapper、Model、Service、Controller层代码
- 内置分页插件：基于MyBatis物理分页
- 内置性能分析插件：可输出SQL语句以及其执行时间

在本系统中，使用MyBatis Plus作为ORM框架，简化了数据库操作，提高了开发效率。

### 2.2 前端技术

#### 2.2.1 Vue.js

Vue.js是一套用于构建用户界面的渐进式JavaScript框架，由尤雨溪于2014年创建。Vue.js的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

Vue.js具有以下特点：
- 响应式数据绑定：使用双向数据绑定，当数据变化时，视图自动更新
- 组件化：将页面拆分成独立的组件，组件可以复用
- 虚拟DOM：使用虚拟DOM提高渲染性能
- 轻量级：核心库压缩后体积小
- 易上手：API简洁，文档齐全

在本系统中，前端采用Vue.js框架，实现了管理员和用户两个前端应用。

#### 2.2.2 Element UI

Element UI是一套基于Vue.js 2.0的组件库，由饿了么前端团队开发。它提供了丰富的UI组件，如按钮、表单、表格、对话框等，能够快速构建美观的Web应用。

Element UI具有以下特点：
- 组件丰富：提供了大量的UI组件，满足各种业务需求
- 易用性：API简洁，文档齐全，易于上手
- 美观：组件设计美观，风格统一
- 可定制：支持主题定制

在本系统中，使用Element UI作为前端UI组件库，快速构建了美观的界面。

### 2.3 数据库技术

#### 2.3.1 MySQL

MySQL是一种关系型数据库管理系统，由瑞典MySQL AB公司开发，现属于Oracle公司。MySQL是一种开放源代码的软件，具有体积小、速度快、总体拥有成本低等特点，被广泛应用于Web开发中。

MySQL具有以下特点：
- 关系型数据库：使用表格存储数据，支持SQL查询
- 开源免费：可以自由使用和修改
- 跨平台：支持Windows、Linux、Mac OS等多种操作系统
- 高性能：支持高并发访问
- 安全：提供了完善的用户权限管理机制

在本系统中，使用MySQL作为数据库，存储系统的所有数据。

#### 2.3.2 Druid

Druid是阿里巴巴开源的一个数据库连接池，是Java语言中最好的数据库连接池之一。Druid能够提供强大的监控和扩展功能，具有性能好、稳定可靠、功能强大等特点。

Druid具有以下特点：
- 性能优秀：相比其他数据库连接池，Druid的性能更好
- 监控功能：提供了强大的监控功能，可以查看SQL执行情况、连接池使用情况等
- 稳定可靠：经过阿里巴巴多年的大规模生产环境验证
- 功能强大：提供了SQL防火墙、SQL执行日志等功能

在本系统中，使用Druid作为数据库连接池，提高了系统的性能和稳定性。

### 2.4 缓存技术

#### 2.4.1 Redis

Redis是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。

Redis具有以下特点：
- 性能极高：读的速度是110000次/s，写的速度是81000次/s
- 丰富的数据类型：支持Strings、Lists、Hashes、Sets、Sorted Sets等数据类型
- 持久化：支持RDB和AOF两种持久化方式
- 高可用：支持主从复制、哨兵模式和集群模式
- 丰富的功能：支持发布订阅、Lua脚本、事务等功能

在本系统中，使用Redis作为缓存，提高了系统的性能。

### 2.5 认证技术

#### 2.5.1 JWT

JWT（JSON Web Token）是一种开放标准（RFC 7519），用于在各方之间安全地传输信息。JWT可以使用HMAC算法或RSA的公私密钥对进行签名，确保信息的安全性。

JWT具有以下特点：
- 简洁：JWT的体积小，便于传输
- 自包含：JWT包含了所有必要的用户信息，不需要额外查询数据库
- 跨语言：JWT是基于JSON的，支持多种语言
- 安全：JWT可以使用密钥进行签名，确保信息的安全性

在本系统中，使用JWT进行身份认证，提高了系统的安全性。

## 4 系统设计

### 4.1 系统架构设计

本系统采用前后端分离的架构，后端使用Spring Boot框架，前端使用Vue.js框架。系统架构分为三层：表现层、业务逻辑层和数据访问层。

#### 4.1.1 表现层

表现层负责接收用户的请求并返回响应。后端表现层使用Spring MVC实现，前端表现层使用Vue.js实现。

#### 4.1.2 业务逻辑层

业务逻辑层负责处理系统的业务逻辑。后端业务逻辑层使用Spring框架实现，提供Service接口和实现类。

#### 4.1.3 数据访问层

数据访问层负责与数据库进行交互。后端数据访问层使用MyBatis Plus实现，提供Mapper接口。

### 4.2 功能模块设计

根据需求分析的结果，本系统分为以下功能模块：

1. **用户认证模块**：负责管理员和顾客的认证、授权和信息管理。
2. **图书管理模块**：负责图书的增删改查、上下架管理和统计。
3. **订单管理模块**：负责订单的创建、支付、处理和退款管理。
4. **捐赠管理模块**：负责捐赠的提交、审核、管理和统计。
5. **购物车模块**：负责购物车的管理。
6. **会员管理模块**：负责会员信息的管理。
7. **库存管理模块**：负责图书库存的管理。
8. **服务反馈模块**：负责用户服务反馈的管理。
9. **系统管理模块**：负责系统用户、角色和菜单的管理。

### 4.4 接口设计

#### 4.4.1 接口设计原则

1. **RESTful风格**：采用RESTful风格的API设计，使用HTTP动词（GET、POST、PUT、DELETE）表示操作类型。
2. **统一返回格式**：所有API接口返回统一格式的响应，包含状态码、消息和数据。
3. **接口版本控制**：支持接口版本控制，便于后续升级。
4. **接口文档完善**：提供完善的接口文档，便于前端开发和后续维护。

#### 4.4.2 主要接口设计

##### 接口4-1 管理员登录接口

- **请求URL**：/api/auth/login
- **请求方法**：POST
- **请求参数**：
  - username：用户名（必填）
  - password：密码（必填）
- **响应数据**：
  - code：状态码
  - message：消息
  - data：

#### 5.1.4 认证模块实现

认证控制器[`AuthController.java`](file:///workspace/houduan/src/main/java/com/shengwei/tushuguanli/controller/AuthController.java)的代码如下：

```java
package com.shengwei.tushuguanli.controller;

import com.shengwei.tushuguanli.common.Result;
import com.shengwei.tushuguanli.entity.SysUser;
import com.shengwei.tushuguanli.service.UserService;
import com.shengwei.tushuguanli.util.JwtUtil;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.web.bind.annotation.*;

import java.util.HashMap;
import java.util.Map;

@RestController
@RequestMapping("/auth")
public class AuthController {

    @Autowired
    private UserService userService;

    @Autowired
    private JwtUtil jwtUtil;

    @Autowired
    private PasswordEncoder passwordEncoder;

    @PostMapping("/login")
    public Result<Map<String, Object>> login(
            @RequestParam String username,
            @RequestParam String password) {
        
        String token = userService.login(username, password);
        SysUser user = userService.getByUsername(username);
        if (user != null) {
            user.setPassword(null);
        }

        Map<String, Object> result = new HashMap<>();
        result.put("token", token);
        result.put("userInfo", user);

        return Result.success("登录成功", result);
    }

    @PostMapping("/logout")
    public Result<Void> logout() {
        userService.logout();
        return Result.success("登出成功");
    }

    @GetMapping("/info")
    public Result<SysUser> getCurrentUser() {
        Long userId = com.shengwei.tushuguanli.config.SecurityContext.getCurrentUserId();
        if (userId == null) {
            return Result.error(401, "未登录");
        }
        SysUser user = userService.getById(userId);
        if (user != null) {
            user.setPassword(null);
        }
        return Result.success(user);
    }

    @PostMapping("/change-password")
    public Result<Void> changePassword(@RequestBody Map<String, Object> params) {
        Long currentUserId = com.shengwei.tushuguanli.config.SecurityContext.getCurrentUserId();
        if (currentUserId == null) {
            return Result.error(401, "未登录");
        }

        String oldPassword = (String) params.get("oldPassword");
        String newPassword = (String) params.get("newPassword");

        if (oldPassword == null || newPassword == null || newPassword.length() < 6) {
            return Result.error("密码格式不正确");
        }

        SysUser user = userService.getById(currentUserId);
        if (user == null) {
            return Result.error("用户不存在");
        }
        if (!passwordEncoder.matches(oldPassword, user.getPassword())) {
            return Result.error("旧密码不正确");
        }
        user.setPassword(passwordEncoder.encode(newPassword));
        userService.updateById(user);
        return Result.success("密码修改成功");
    }
}
```

```java
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/book")
public class BookController {

    @Autowired
    private BookInfoService bookInfoService;

    @GetMapping("/page")
    public Result<PageResult<BookInfo>> pageBookList(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize,
            String bookName,
            String author,
            String isbn,
            String publisher,
            Long categoryId,
            Integer shelfStatus,
            Integer isHot,
            Integer isNew,
            Integer isDonation,
            String sortBy) {
        
        Map<String, Object> params = new HashMap<>();
        params.put("bookName", bookName);
        params.put("author", author);
        params.put("isbn", isbn);
        params.put("publisher", publisher);

        params.put("isHot", isHot);
        params.put("isNew", isNew);
        params.put("isDonation", isDonation);
        params.put("sortBy", sortBy);

        PageResult<BookInfo> result = PageResult.of(
                page.getTotal(),
                page.getPages(),
                page.getCurrent(),
                page.getSize(),
                page.getRecords()

        return Result.success(result);
```

#### 5.1.6 订单管理模块实现

订单控制器[`TradeOrderController.java`](file:///workspace/houduan/src/main/java/com/shengwei/tushuguanli/controller/TradeOrderController.java)的代码如下：

```java
package com.shengwei.tushuguanli.controller;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.shengwei.tushuguanli.common.Result;
import com.shengwei.tushuguanli.entity.CartItemVO;
import com.shengwei.tushuguanli.entity.TradeOrder;
import com.shengwei.tushuguanli.entity.TradeOrderItem;
import com.shengwei.tushuguanli.entity.TradeOrderRefund;
import com.shengwei.tushuguanli.mapper.TradeOrderItemMapper;
import com.shengwei.tushuguanli.service.ShoppingCartService;
import com.shengwei.tushuguanli.service.TradeOrderService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/order")
public class TradeOrderController {

    @Autowired
    private TradeOrderService orderService;

    @Autowired
    private ShoppingCartService cartService;

    @Autowired
    private TradeOrderItemMapper orderItemMapper;

    @PostMapping("/create")
    public Result<TradeOrder> createOrder(@RequestBody Map<String, Object> params) {
        Long currentUserId = com.shengwei.tushuguanli.config.SecurityContext.getCurrentUserId();
        if (currentUserId == null) {
            return Result.error(401, "请先登录");
        }
        List<CartItemVO> cartItems = cartService.getCartList(currentUserId);
        if (cartItems.isEmpty()) {
            return Result.error("购物车为空");
        }
        TradeOrder order = orderService.createOrder(currentUserId, cartItems);
        return Result.success(order);
    }

    @PostMapping("/pay")
    public Result<Void> payOrder(@RequestBody Map<String, Object> params) {
        String orderNo = params.get("orderNo").toString();
        orderService.payOrder(orderNo);
        return Result.success("支付成功");
    }

    @GetMapping("/list")
    public Result<List<TradeOrder>> getOrderList() {
        Long currentUserId = com.shengwei.tushuguanli.config.SecurityContext.getCurrentUserId();
        if (currentUserId == null) {
            return Result.error(401, "请先登录");
        }
        List<TradeOrder> orderList = orderService.getOrderList(currentUserId);
        return Result.success(orderList);
    }

    @GetMapping("/items/{orderId}")
    public Result<List<TradeOrderItem>> getOrderItems(@PathVariable Long orderId) {
        List<TradeOrderItem> items = orderItemMapper.selectList(
                new LambdaQueryWrapper<TradeOrderItem>().eq(TradeOrderItem::getOrderId, orderId));
        return Result.success(items);
    }

    @GetMapping("/all")
    public Result<List<Map<String, Object>>> getAllOrders(
            @RequestParam(required = false) String orderNo,
            @RequestParam(required = false) Integer status) {
        List<Map<String, Object>> orders = orderService.getAllOrders(orderNo, status);
        return Result.success(orders);
    }

    @GetMapping("/detail/{id}")
    public Result<Map<String, Object>> getOrderDetail(@PathVariable Long id) {
        Map<String, Object> detail = orderService.getOrderDetail(id);
        return Result.success(detail);
    }

    @PutMapping("/ship/{orderId}")
    public Result<Void> shipOrder(@PathVariable Long orderId) {
        orderService.shipOrder(orderId);
        return Result.success("发货成功");
    }

    @PutMapping("/complete/{orderId}")
    public Result<Void> completeOrder(@PathVariable Long orderId) {
        orderService.completeOrder(orderId);
        return Result.success("订单已完成");
    }

    @PutMapping("/cancel/{orderNo}")
    public Result<Void> cancelOrder(@PathVariable String orderNo) {
        orderService.cancelOrder(orderNo);
        return Result.success("订单已取消");
    }

    @PostMapping("/refund/apply")
    public Result<Void> applyRefund(@RequestBody Map<String, Object> params) {
        String orderNo = params.get("orderNo").toString();
        String reason = params.get("reason").toString();
        Long currentUserId = com.shengwei.tushuguanli.config.SecurityContext.getCurrentUserId();
        if (currentUserId == null) {
            return Result.error(401, "请先登录");
        }
        orderService.applyRefund(orderNo, currentUserId, reason);
        return Result.success("退款申请已提交");
    }

    @GetMapping("/refund/list")
    public Result<List<TradeOrderRefund>> getRefundList(
            @RequestParam(required = false) Long userId,
            @RequestParam(required = false) String orderNo,
            @RequestParam(required = false) Integer status) {
        List<TradeOrderRefund> refundList = orderService.getFilteredRefundList(userId, orderNo, status);
        return Result.success(refundList);
    }

    @PostMapping("/refund/handle")
    public Result<Void> handleRefund(@RequestBody Map<String, Object> params) {
        Object refundIdObj = params.get("refundId");
        Object statusObj = params.get("status");
        if (refundIdObj == null || statusObj == null) {
            return Result.error(400, "缺少必要参数");
        }
        try {
            Long refundId = Long.valueOf(refundIdObj.toString());
            Integer status = Integer.valueOf(statusObj.toString());
            String remark = params.get("remark") != null ? params.get("remark").toString() : null;
            orderService.handleRefund(refundId, status, remark);
            return Result.success("处理成功");
        } catch (NumberFormatException e) {
            return Result.error(400, "参数格式错误");
        }
    }
}
```
