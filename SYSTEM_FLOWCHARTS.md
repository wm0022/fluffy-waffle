## 3. 订单业务流程图

```mermaid
flowchart TD
    A[顾客登录] --> B[浏览图书]
    B --> C[加入购物车]
    C --> D{购物车是否为空?}
    D -->|是| B
    D -->|否| E[提交订单]
    E --> F[生成订单]
    F --> G[订单待支付]
    G --> H[支付订单]
    H --> I{支付成功?}
    I -->|否| G
    I -->|是| J[订单已支付]
    J --> K[管理员发货]
    K --> L[订单已发货]
    L --> M[顾客确认收货]
    M --> N{确认收货?}
    N -->|否| L
    N -->|是| O[订单已完成]
    
    G --> P[取消订单]
    P --> Q[订单已取消]
    
    J --> R[申请退款]
    R --> S[退款审核中]
    S --> T{审核通过?}
    T -->|否| U[退款驳回]
    T -->|是| V[退款成功]
```

## 5. 库存管理流程图

